# StableHacks Submission Checklist

Use this checklist before pressing final submit on DoraHacks.

## Team Metadata
- [ ] Project name: `CompliPay AI`
- [ ] Team member list is complete
- [ ] Country field is complete

## Required Links
- [ ] Public GitHub repository URL
- [ ] Loom video URL (max 3 minutes)
- [ ] Testnet demo URL or technical overview video URL

## Product Validation
- [ ] At least one payment execution flow demonstrated

- [ ] Audit trail evidence shown in demo

- [ ] Demo dry-run #3 completed without blocker

## Final Submission
- [ ] All fields reviewed for accuracy
